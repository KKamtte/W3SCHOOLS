jQuery Syntax For Event Methods

jQuery에서 대부분의 DOM 이벤트에는 동일한 jQuery 메소드가 있다.
페이지의 모든 단락에 클릭 이벤트를 할당하려면 다음과 같이 하면된다.

        $("p").click();

다음 단계로 이벤트가 발생할 때 어떤 일이 발생해야하는지 정의하는 것이다.
이벤트에 함수를 전달해야한다.

    $("p").click(function(){
        //이벤트 정의
    });

Commonly Used jQuery Event Methods

$(document).ready()

$(document).ready() 메소드를 사용하면 문서가 완전히 로드 되었을 때 함수를 실행 할 수 있다.

click()

click() 메소드는 이벤트 핸들러 함수를 HTML 요소에 연결한다.
사용자가 HTML 요소를 클릭하면 함수가 실행된다.
다음 예제는 p태그에 클릭 이벤트가 발생하는 경우 현재 p요소를 숨긴다.

        $("p").click(function(){
            $(this).hide();
        });

dblclick()

dblclick() 메소드는 이벤트 핸들러 함수를 HTML 요소에 연결한다.
이 함수는 사용자가 HTML 요소를 두 번 클릭하면 실행된다.

        $("p").dblclick(function(){
            $(this).hide();
        });

mouseenter()

mouseenter() 메소드는 이벤트 핸들러 함수를 HTML 요소에 연결한다.
이 함수는 마우스 포인터가 HTML 요소에 들어갈 때 실행된다.

        $("#p1").mouseenter(function(){
            alert("p1에 마우스를 올렸다.");
        });

mouseleave()

mouseleave() 메소드는 이벤트 핸들러 함수를 HTML 요소에 연결한다.
이 함수는 마우스 포인터가 HTML 요소를 떠날 때 실행된다.

        $("#p1").mouseleave(function(){
            alert("p1에 마우스를 벗어났다.");
        });

mousedown()

mousedown() 메소드는 이벤트 핸들러 함수를 HTML 요소에 연결한다.
이 함수는 마우스 왼쪽, 가운데, 오른쪽 버튼을 누른 상태에서 마우스가 HTML 요소 위에 있을 때 실행된다.

        $("#p1").mousedown(function(){
            alert("p1을 클릭하였다.");
        });

mouseup()

mouseup() 메소드는 이벤트 핸들러 함수를 HTML 요소에 연결한다.
마우스가 HTML 요소 위에 있는 동안 왼쪽, 가운데, 오른쪽 버튼을 놓으면 실행된다.

        $("#p1").mouseup(function(){
            alert("p1을 클릭하였다.");
        });

hover()

hover() 메소드는 mouseenter과 mouseleave 메소드를 합친 것이다.
첫 번째 함수는 마우스가 HTML 요소에 들어갈 때 실행되고 두 번째 함수는 마우스가 HTML 요소를 떠날 때 실행된다.

        $("#p1").hover(function(){
            alert("p1을 들어왔다.");
        },
        function(){
            alert("p1을 떠났다.");
        });

foucs()

focus() 메소드는 필드에 포커스가있을 때 실행된다.

        $("input").focus(function(){
            $(this).css("background-color", "#cccccc");
        });

blur()

blur() 메소드는 필드가 포커스를 잃을 때 실행된다.

        $("input").blur(function(){
            $(this).css("background-color", "#ffffff");
        });

The on() Method

on() 메소드는 선택한 요소에 대해 하나 이상의 이벤트 핸들러를 연결한다.

        $("p").on({
            mouseenter: function(){
                $(this).css("background-color", "lightgray");
            },
            mouseleave: function(){
                $(this).css("background-color", "lightblue");
            },
            click : function(){
                $(this).css("background-color", "yellow");
            }
        });
