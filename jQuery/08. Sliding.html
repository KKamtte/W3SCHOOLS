jQuery Sliding Methods

jQuery를 사용하면 요소에 슬라이딩 효과를 만들 수 있다.
jQuery에는 다음과 같은 slide 메소드가 있다.

  • slideDown()
  • slideUp()
  • slideToggle()

jQuery slideDown() Method

Syntax :
$(selector).slideDown(speed, callback);

slidedown panel 실행
Hello World!

jQuery slideUp() Method

Syntax :
$(selector).slideUp(speed, callback);

slideUp panel 실행
Hello World!

jQuery slideToggle() Method

Syntax :
$(selector).slideToggle(speed, callback);

slideToggle panel 실행
Hello World!
