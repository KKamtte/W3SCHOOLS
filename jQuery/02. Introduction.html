What is jQuery?

jQuery는 가볍고 "덜 쓰고 더 많은 작업을 수행"하는 JavaScript 라이브러리이다.
jQuery의 목적은 웹 사이트에서 JavaScript를 훨씬 더 쉽게 사용하도록 하는 것 이다.
jQuery는 여러 줄의 JavaScriptCode가 필요한 많은 일반적인 작업을 수행하고 한 줄의 코드로 호출 할 수있는 메소드로 래핑한다.
jQuery는 또한 AJAX 호출 및 DOM 조작과 같은 JavaScript의 복잡한 많은 것들을 단순화한다.
jQuery라이브러리에는 다음과 같은 기능이 포함되어있다.

  • HTML/DOM 조작
  • CSS 조작
  • HTML 이벤트 메소드
  • 효과 및 애니메이션
  • AJAX
  • Utilites

또한 jQuery에는 거의 모든 작업을 위한 플러그인이 있다.

Why jQuery?

다른 많은 JavaScript라이브러리가 있지만 jQuery가 가장 인기있고 확장성이 뛰어나다.
웹 상의 다음과 같은 많은 대기업들이 jQuery를 사용한다.

  • Google
  • Microsoft
  • IBM
  • Netflix

Will jQuery work in all browsers?

jQuery팀은 브라우저 간 문제에 대해 모두 알고 있으며 이 지식을 jQuery라이브러리에 기록하였다.
jQuery는 모든 주요 브라우저에서 정확히 동일하게 실행된다.
