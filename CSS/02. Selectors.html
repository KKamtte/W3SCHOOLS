CSS Selectors

CSS selectors는 스타일을 지정할 HTML 요소를 찾거나 선택하는데 사용된다.

CSS selectors five categories

  • Simple selectors(이름, ID, 클래스를 기준으로 요소 선택)
  • Combinator selectors(특정 관계에 따라 요소 선택)
  • Pseudo-class selectors(특정 상태에 따라 요소 선택)
  • Pseudo-elements selectors(요소의 일부 선택 및 스타일 지정)
  • Attribute selectors(속성 또는 속성 값에 따라 요소 선택)

The CSS element Selector

element selector는 요소 이름을 기반으로 HTML 요소를 선택한다.

Every paragraph will be affected by the style.
Me too!
And me!

The CSS id Selector

id selector는 id속성을 사용하여 특정 요소를 선택한다.

Hello World!
This paragraph is not affected by the style.

The CSS class Selector

class selector는 .을 이용하여 요소를 선택한다
element selector와 결합하여 사용할 수 있다

This heading is center class

This paragraph is center class
