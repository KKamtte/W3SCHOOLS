Using let and const

자바스크립트에서 var라는 변수만 사용되었지만 2015년 이후 const키워드를 사용하여 재 할당 없는 변수를 정의하고 let키워드를 사용하여 제한된 범위의 변수를 정의 할 수 있게되었다.

JavaScript Data Types

자바스크립트는 100과 같은 숫자와 "john"과 같은 텍스트 값을 포함 할 수 있다.
문자열은 큰 따옴표 또는 작은 따옴표 안에 작성되고 숫자는 따옴표 없이 작성된다
따옴표 안에 숫자를 넣으면 텍스트 문자열로 처리가 된다.

JavaScript Dolloar Sign $

JavaScript 식별자(이름)는 다음으로 시작해야 한다.

  • 문자(A-Z or a-z)
  • 달러사인($)
  • 언더스코어(_)
