JavaScript Date Input

일반적으로 3가지 유형의 JavaScript 날짜 입력 형식이 있다.

Type Example
ISO Date "2015-03-25" (The International Standard)
Short Date "03/25/2015"
Long Date "Mar 25 2015" or "25 Mar 2015"

ISO 형식은 JavaScript의 엄격한 표준을 따른다.
다른 형식은 잘 정의되어 있지 않으며 브라우저에 따라 다를 수 있다.

JavaScript Date Output

입력 형식과 관계없이 JavaScript는 전체 텍스트 문자열 형식으로 날짜를 출력한다.

JavaScript ISO Dates

ISO 8601은 날짜 및 시간표현에 대한 국제 표준이다.
ISO 8601 구문(YYYY-MM-DD)도 선호되는 JavaScript 날짜 형식이다.

ISO Dates (Year and Month)

ISO 날짜는 날짜(YYYY-MM)를 지정하지 않고 쓸 수 있다.

ISO Dates (Only Year)

ISO 날짜는 월과 일(YYYY)없이 작성할 수 있다.

ISO Dates (Date-Time)

ISO 날짜는 추가 된 시간, 분 및 초(YYYY-MM-DDTHH:MM:SSZ)로 작성 할 수 있다.

날짜와 시간은 대문자 T로 구분된다.
UTC시간은 대문자 Z로 정의된다.
UTC를 기준으로 시간을 수정하려면 Z를제거하고 대신 +HH:MM또는 -HH:MM을 추가한다.

Time Zones

시간대를 지정하지 않고 날짜를 설정할 때 JavaScript는 브라우저의 시간대를 사용한다.
시간대를 지정하지 않고 날짜를 가져오면 결과가 브라우저의 시간대로 변환된다.

JavaScript Short Dates

짧은 날짜는 "MM/DD/YYYY"구문으로 작성된다.

JavaScript Long Dates

긴 날짜는 "MMM DD YYYY"구문으로 작성된다.

Date Input - Parsing Dates

유효한 날짜 문자열이 있는 경우 Date.parse() 메소드를 사용하여 밀리초로 변환할 수 있다.
Date.parse()는 날짜와 1970년 1월 1일 사이의 밀리 초 수를 반환한다.
