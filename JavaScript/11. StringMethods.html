String Length

length 는 문자열의 길이를 출력해준다

Finding a String in a String

indexOf() 는 문자열에서 지정된 텍스트의 첫 번째 위치를 반환한다.

lastIndexOf() 는 문자열에서 지정된 텍스트의 마지막 위치를 반환한다.

만약 해당하는 문자를 찾지 못한 경우 -1을 반환한다.

두 번째 파라미터에 숫자를 넣으면 해당 지점부터 찾는 것을 시작한다.

Searching for a String in a String

search() 는 문자열에서 지정된 값을 검색하고 일치하는 위치를 반환한다.

indexOf() 와 search()의 차이점

  • search() 메소드는 두번째 시작 위치 인수를 사용할 수 없다.
  • indexOf() 메소드는 정규식을 사용할 수 없다.
