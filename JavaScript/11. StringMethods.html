String Length

length 는 문자열의 길이를 출력해준다

Finding a String in a String

indexOf() 는 문자열에서 지정된 텍스트의 첫 번째 위치를 반환한다.

lastIndexOf() 는 문자열에서 지정된 텍스트의 마지막 위치를 반환한다.

만약 해당하는 문자를 찾지 못한 경우 -1을 반환한다.

두 번째 파라미터에 숫자를 넣으면 해당 지점부터 찾는 것을 시작한다.

Searching for a String in a String

search() 는 문자열에서 지정된 값을 검색하고 일치하는 위치를 반환한다.

indexOf() 와 search()의 차이점

  • search() 메소드는 두번째 시작 위치 인수를 사용할 수 없다.
  • indexOf() 메소드는 정규식을 사용할 수 없다.

Extracting String Parts

문자열의 일부를 추출하는 방법에는 세 가지가 있다.

  • slice(start, end)
  • substring(start, end)
  • substr(start, length)

The slice() Method

slice()는 문자열의 일부를 추출하고 추출 된 부분을 새 문자열로 반환한다.
이 메소드는 시작위치와 끝위치 두개의 메소드를 받는다

시작위치만 적게 되면 끝 위치는 자동으로 문자열의 끝까지 반환된다.

The substring() Method

substring()은 slice()와 비슷하지만 음수 인덱스를 허용하지 않는다.

The substr() Method

substr()은 두번째 매개변수가 추출된 부분의 길이를 지정한다.

Replacing String Content

replace()는 메소드의 지정된 문자를 다른 값으로 바꿔준다.

대소문자를 구분하지 않으려면 정규식에 /i 플래그(비 구분)와 함께 사용하면 된다.

모든 일치 항목을 바꾸려면 정규식을 /g 플래그(전역일치)와 함께 사용하면 된다.
