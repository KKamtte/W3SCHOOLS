Set Date 메소드를 사용하면 Date 객체에 대한 날짜 값(년, 월, 일, 시, 분, 초, 밀리초)를 설정할 수 있다.

Set Date Methods

다음의 메소드는 날짜 개체에서 일부를 설정하는데 사용된다.

Method Description
setDate() 날짜를 숫자로 설정(1-31)
setFullYear() 연도 설정(선택적으로 월과 일 설정)
setHours() 시간 설정(0-23)
setMilliseconds() 밀리 초 설정(0-999)
setMinutes() 분 설정(0-59)
setMonth() 월 설정(0-11)
setSeconds() 초 설정(0-59)
setTime() 시간 설정(1970년1월1일 이후 밀리초)

The setFullYear() Method

setFullYear() 메소드는 날짜 객체의 연도를 설정한다.

The setMonth() Method

setMonth() 메소드는 날짜 객체의 월(0-11)을 설정한다.

The setDate() Method

setDate() 메소드는 날짜 객체의 일(1-31)을 설정한다.

The setHours() Method

setHours() 메소드는 날짜 객체의 시간(0-23)을 설정한다.

The setMinutes() Method

setMinutes() 메소드는 날짜 객체의 분(0-59)을 설정한다.

The setSeconds() Method

setSeconds() 메소드는 날짜 객체의 초(0-59)를 설정한다.

Compare Dates

날짜를 비교하는것은 단순히 비교 연산자만 사용하면 된다.
