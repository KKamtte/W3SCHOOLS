JavaScript Display Possibilites

JavaScript에서는 다양한 방식으로 출력할 수 있다.

  • innerHTML
  • document.write()
  • window.alert()
  • console.log()

Using innerHTML

document.getElementById('id').innerHTML = 5 + 6;

Using document.write()

document.write(5 + 6);

<button type="button" onclick="document.write(5 + 6)">Try it</button>

Using window.alert()

window.alert(5 + 6)
window 생략가능
alert("alert")

Using console.log()

console.log() 는 디버깅할때 사용됨

JavaScript Print
