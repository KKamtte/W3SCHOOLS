Math.PI

Pi 값을 반환해준다.

Math.round()

Math.round(x)는 가장 가까운 정수로 반올림된 x값을 반환한다.

Math.pow()

Math.pow(x,y)는 x의 값을 y의 거듭 제곱으로 반환한다.

Math.sqrt()

Math.sqrt(x) 는 x의 루트값을 반환한다.

Math.abs()

Math.abs(x) 는 x의 절대값을 반환한다.

Math.ceil()

Math.ceil(x) 는 x의 올림값을 반환한다.

Math.floor()

Math.floor(x) 는 x의 내림값을 반환한다.

Math.sin()

Math.sin(x) 는 각도 x의 사인값을 반환한다.
라디안을 사용하려면 변환을 해야 한다.
라디안 각도 = 각도 X PI / 180

Math.cos()

Math.cos(x) 는 각도 x의 코사인값을 반환한다.

Math.min() and Math.max()

Math.min()과 Math.max()를 사용하여 목록에서 가장 낮은 값과 높은 값을 찾을 수 있다.

Math.random()

Math.random()은 0-1 사이의 값을 반환한다.

Math Properties(Constants)

JavaScript 는 Math 객체로 접근할 수 있는 8개의 수학 상수를 제공한다.
