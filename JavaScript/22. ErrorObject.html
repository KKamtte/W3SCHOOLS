JavaScript 에는 오류 발생 시 오류 정보를 제공하는 오류 개체가 내장되어 있다.
오류 개체는 이름과 메시지 라는 두 가지 속성을 제공한다.

Error Object Properties

Property Description
name 오류 이름을 설정하거나 반환한다.
message 오류 메시지(문자열)를 설정하거나 반환한다.

Error Name Values

오류 이름 속성은 여섯 가지 값을 반환할 수 있다.

Error Name Description
EvalError eval() 함수에서 오류가 발생하였다.
RangeError 범위를 벗어난 숫자가 발생하였다.
ReferenceError 잘못된 참조가 발생하였다.
SyntaxError 구문 오류가 발생하였다.
TypeError 유형 오류가 발생하였다.
URIError encodeURI() 에서 오류가 발생하였다.

Eval Error

eval() 함수의 오류를 나타낸다.
최신 버전의 JavaScript 는 EvalError를 발생시키지 않는다.
대신 SyntaxError 를 사용해야한다.

Range Error

유효한 값 범위를 벗어난 숫자를 사용하면 RangeError가 발생한다.

Reference Error

선언되지 않은 변수를 사용(참조)하면 ReferenceError가 발생한다.

Syntax Error

구문 오류가 있는 코드를 사용하려고 하면 SyntaxError가 발생한다.

Type Error

예상 유형 범위를 벗어난 값을 사용하면 TypeError가 발생한다.

URI (Uniform Resource Identifier) Error

URI 함수에 잘못된 문자를 사용하면 URIError가 발생한다.
