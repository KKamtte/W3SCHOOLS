JavaScript Function Syntax

JavaScript 함수는 키워드,이름,괄호로 정의된다.
함수 이름에는 문자, 숫자, 밑줄 및 달러 기호(변수와 동일)가 포함될 수 있다.
괄호에는 쉼표로 구분된 매개 변수 이름이 포함 될 수 있다.(매개변수1, 매개변수2)
함수에 의해 실행될 코드는 중괄호 안에 있다.

function name(parameter1, parameter2, parameter3) {
    // code to be executed
}

Function Invocation

함수 내부의 코드는 함수를 호출할때 실행이 된다

  • 이벤트 발생시(사용자 버튼 클릭시))
  • JavaScript코드에서 호출되는 경우
  • 자동(자체)호출

Function Return

JavaScript가 return 문에 도달하면 함수 실행이 중지된다.
함수가 명령문에서 호출 된 경우 JavaScript는 호출 명령문 다음에 코드를 실행하기 위해 return을 한다.
함수는 종종 return 값을 계산하고 return 값은 발신자에게 return 된다.

The () Operator Invokers the Function

() 없이 함수를 액세스하면 함수 결과 대신 함수 객체가 반환된다.

Function Used as Variable Values

함수는 모든 유형의 공식, 할당 및 계산에서 변수를 사용하는 것과 동일한 방식으로 사용될 수 있다.

Local Variables

JavaScript 함수 내에서 선언된 변수는 함수에 대해 지역변수가 된다.
지역 변수는 함수 내에서만 접근가능하다.
