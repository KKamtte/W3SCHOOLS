Creating Date Objects

Date 객체는 new Date()생성자로 생성된다.
새 날짜 개체를 만드는 방법에는 4가지가 있다.

  • new Date()
  • new Date(year, month, day, hours, minutes, seconds, milliseconds)
  • new Date(milliseconds)
  • new Date(date string)

new Date()

new Date() 는 현재 날짜와 시간으로 새 날짜 객체를 만든다.

new Date(year, month, ...)

new Date(year, month, ...)는 지정된 날짜와 시간으로 새 날짜 객체를 만든다.
7개의 숫자는 년, 월, 일, 시, 분, 초, 밀리초를 지정한다.

JavaScript에서는 0~11까지 숫자를 이용해서 월을 나타낸다.
1월은 0이고 12월은 11이다.

월을 생략하는 경우 밀리초로 나타나게 된다.

Previous Century

한 자리 및 두 자리 연도는 19XX년으로 해석된다.

new Date(dateString)

new Date(dateString)는 날짜 문자열에서 새 날짜 객체를 만든다.

Displaying Dates

toUTCString() 메소드는 날짜를 UTC문자열(표준시)로 변환한다.

toDateString() 메소드는 날짜를 더 읽기 쉬운 형식으로 변환한다.

toISOString() 메소드는 ISO 표준 형식을 사용하여 날짜를 문자열로 변환한다.
