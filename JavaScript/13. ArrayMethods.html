Converting Arrays to Strings

배열에 .toString()으로 출력하면 ,로 구분된 문자열로 출력된다.

배열에 .join()으로 출력하면 구분값을 만들어서 출력할 수 있다.

Popping and Pushing

pop()을 이용하면 배열의 마지막 요소가 제거된다.

push()를 이용하면 배열의 마지막에 새 요소를 추가한다.

Shifting Elementes

shift()는 첫 번째 배열의 요소를 제거하고 뒤 요소 배열을 앞으로 당겨온다.

unshift()는 첫 번째 배열 요소에 값을 추가한다.

Changing Elements

배열의 인덱스 요소접근을 통해 각 요소의 값을 변경할 수 있다.

length를 이용하면 배열의 마지막 요소에 값을 추가한다.

Deleting Elements

delete를 이용하여 배열의 요소를 제거할 수 있다.

Splicing an Array

splice()를 이용하면 배열에 새 항목을 추가할 수 있다.

splice의 첫번째 입력값은 어디에 추가될 것인지를 의미한다.
두번째 입력값은 몇개를 지울 것인지 결정한다

Using splice() to Remove Elements

splice()를 이용하면 배열에 빈 값을 남기지 않고 삭제할 수 있다.

Merging (Concatenating) Arrays

concat()를 이용하여 배열을 합칠 수 있다.

concat배열에 value값을 추가하는 방식

Slicing an Array

slice()는 배열의 일부를 새 배열로 분할한다.

Automatic toString()

배열 요소를 String으로 형 변환을 하기 위해서 .toString()을 사용한다.
