JavaScript Data Types

The Concept of Data Types

data type은 중요한 개념이다. 변수에 대해 작동할 수 있으려면 type에 대해 하는 것이 중요하다.

JavaScript Types are Dynamic

JavaScript에는 동적 유형이 있다. 즉, 동일한 변수를 사용하여 다른 데이터 유형을 보유 할 수 있다.

JavaScript Arrays

배열 선언은 var cars = ["bmw", "benz", "volvo"] 형태로 선언한다

JavaScript Objects

Object 선언은 var person = {firstName:"Kim", lastName:"SeungHyeon", age:27, eyeColor:"black"} 형태로 선언한다

Undefined

JavaScript에서 값이 없는 변수는 정의되지 않는 값을 갖는다. 유형또한 정의되지 않는다.

Empty Values

빈 값(empty value)은 정의되지않은 값(undefined)과 다르다. 빈 값은 string유형을 갖는다.

Null

JavaScript에서 null은 "nothing"이다. 그것은 존재하지 않는 값이다.
null의 데이터 유형은 객체이다. null을 통해 객체를 비울 수 있다.
