Array.forEach()

forEach() 메소드는 각 배열 요소에 대해 한 번씩 함수를 호출한다.

Array.map()

map() 메소드는 각 배열요소에 대해 함수를 수행하여 새 배열을 만든다.
map() 메소드는 값이 없는 배열 요소에 대해 함수를 실행하지 않는다.
원래 배열을 변경하지 않는다.

Array.filter()

filter() 메소드는 조건에 부합하는 요소로 새 배열을 만든다.

Array.reduce()

reduce() 메소드는 각 배열 요소에서 함수를 실행하여 단일 값을 생성한다.
reduce() 메소드는 배열의 왼쪽에서 오른쪽으로 작동한다.

reduce() 메소드는 초기값을 받을 수 있다.

Array.every()

every() 메소드는 모든 배열의 값이 테스트를 통과하는지 확인한다.(True, False 반환)

Array.some()

some() 메소드는 일부 배열의 값이 테스트를 통과하는지 확인한다.(True, False 반환)

Array.indexOf()

indexOf() 메소드는 배열에서 요소 값을 검색하고 해당 위치를 반환한다.(앞에서부터 진행)

Array.lastIndexOf()

lastIndexOf() 메소드는 배열의 뒤 부터 요소 값을 검색하고 해당 위치를 반환한다.

Array.find()

find() 메소드는 테스트 함수를 통과하는 첫 번째 배열 요소의 값을 반환한다.

Array.findIndex()

findIndex() 메소드는 배열 요소의 위치를 반환한다.
