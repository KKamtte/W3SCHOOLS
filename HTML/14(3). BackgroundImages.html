거의 모든 HTML요소에 대해 배경 이미지를 지정할 수 있다.

Background Image on a HTML element

<div style="background-image : url('img.jpg');">

You can specify background images
for any visible HTML element.
In this example, the background image
is specified for a div element.
By default, the background-image
will repeat itself in the direction(s)
where it is smaller than the element
where it is specified. (Try resizing the
browser window to see how the
background image behaves.

Background Image on a Page - 예제

image파일이 웹 페이지 전체에 추가하고 싶은 경우 <body> 부분에 넣으면된다.

Background Repeat - 예제(no-repeat)

배경 이미지가 요소보다 작으면 요소 끝에 도달할때까지 이미지가 가로 및 세로로 반복된다.
이 과정을 막으려면 background-repeat 속성을추가하여 no-repeat으로 정의하면 된다.

Background Cover - 예제

배경 이미지가 전체 요소를 덮도록 하려면 background-size 속성을 cover로 설정하면된다.
또한 전체 요소가 항상 덮여있는지 확인하려면 background-attachment 속성을 fixed로 설정하면 된다.
이렇게하면 배경 이미지가 늘어나지않고 전체 요소를 덮는다.

Background Stretch - 예제

배경 이미지를 전체 요소에 맞게 늘리려면 background-size 속성을 100% 100%로 설정하면 된다.
