Ordered HTML List - The Type Attribute

<ol> 태그의 type 속성은 목록 항목 마커의 유형을 정의한다.

Type Description
type="1" 숫자 번호(기본값)
type="A" 대문자 번호
type="a" 소문자 번호
type="I" 대문자 로마 숫자
type="i" 소문자 로마 숫자

Numbers:

  1. Coffee
  2. Tea
  3. Milk

Uppercase Letters:

  1. Coffee
  2. Tea
  3. Milk

Lowercase Letters:

  1. Coffee
  2. Tea
  3. Milk

Uppercase Roman Numbers:

  1. Coffee
  2. Tea
  3. Milk

Lowercase Roman Numbers:

  1. Coffee
  2. Tea
  3. Milk

Control List Counting

기본적으로 정렬 된 목록은 1부터 계산된다.
지정된 숫자부터 계산을 시작할면 start 속성을 사용하면 된다.
