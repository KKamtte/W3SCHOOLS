HTML paragraphs

<p> 요소는 paragraph로 정의되어있다.

항상 새 줄에서 시작하며 브라우저는 단락 앞뒤에 공백을 자동으로 추가한다.

HTML Display

HTML을 사용하면 공백이나 줄을 추가하여 표시를 변경할 수 없다.
브라우저는 페이지가 표시될 때 추가 공백과 줄을 자동으로 제거한다.

This paragraph contains a lot of lines in the source code, but the browser ignores it.

This paragraph contains a lot of spaces in the source code, but the browser ignores it.

HTML Horizontal Rules

<hr> 태그는 페이지에서 주제별 구분을 정의하며 가로줄이 생성된다.
콘텐츠를 분리(또는 변경,정의)하는 데 사용된다.

This is heading 1

This is some text.

This is heading 2

This is some other text.

HTML Line Breaks

<br> 요소는 줄 바꿈에 사용된다.

This is
a paragraph
with line breaks

The Poem Problem

<p> 태그를 이용하면 여러줄의 문장이 한줄로 표시된다.

My Bonnie lies over the ocean. My Bonnie lies over the sea. My Bonnie lies over the ocean. Oh, bring back my Bonnie to me.

Solution - The HTML <pre> Element

<pre> 요소는 미리 서식이 지정된 텍스트를 정의한다.
<pre> 요소 내부의 텍스트는 고정 너비 글꼴로 표시되며 공백과 줄 바꿈을 모두 유지한다.
  	My Bonnie lies over the ocean.

  	My Bonnie lies over the sea.

  	My Bonnie lies over the ocean.

  	Oh, bring back my Bonnie to me.
