The HTML Style Attribute

HTML 요소의 style을 세팅하면 style 속성이 적용된다.
style속성은 다음 구문을 따른다

<tagname style="property:value;">

property와 value 는 CSS의 property와 value이다.

Background Color

CSS의 background-color 속성은 HTML의 backgound 요소를 정의한다.
	<body style="background-color:powderblue;">

	<h1>This is a heading</h1>
	<p>This is a paragraph.</p>

	</body>
