링그는 거의 모든 웹 페이지에서 찾을 수 있다. 링크를 통해 사용자는 페이지간에 클릭 할 수 있다.

HTML Links - Hyperlinks

HTML 링크는 hyperlinks 이다.
링크를 클릭하여 다른 문서로 이동할 수 있다.
링크 위로 마우스를 이동하면 마우스 화살표가 작은 손 모양으로 바뀌게 된다.

Note: 링크는 텍스트일 필요가 없다. 링크는 이미지 또는 기타 HTML 요소가 될 수 있다.

HTML Links - Syntax

HTML <a> 태그는 하이퍼 링크를 정의한다. 구문은 다음과 같다

<a href="url">link text</a>

<a> 요소의 가장 중요한 속성은 링크의 목적지를 나타내는 href 속성이다.
링크 텍스트는 웹페이지를 보는 사람이 볼 수 있는 부분이다.
링크 텍스트를 클릭하면 리더가 지정된 URL 주소로 전송된다.

예제 링크 - Visit W3Schools.com!

기본적으로 링크는 모든 브라우저에서 다음과 같이 나타난다.

  • 방문하지 않은 링크는 밑줄과 파란색으로 표시된다.
  • 방문한 링크는 밑줄과 자주색으로 표시된다.
  • 활성 링크는 밑줄과 빨간색으로 표시된다.

Tip: 링크는 CSS로 스타일을 지정하여 다른 모습으로 표시할 수 있다.

HTML Links - The target Attribute

기본적으로 링크 된 페이지는 현재 브라우저 창에 표시된다. 이를 변경하려면 링크에 대해 다른 대상을 지정해야한다.

target 속성은 링크 된 문서를 열 위치를 지정한다.

target 속성은 다음 값 중 하나를 가질 수 있다.

  • _self - 기본값. 클릭한 것과 동일한 창/탭 에서 문서를 연다.
  • _blank - 새 창 또는 탭에서 문서를 연다.
  • _parent - 상위 프레임에서 문서를 연다.
  • _top - 창의 전체 본문에서 문서를 연다.

<a href="https://www.w3schools.com/" target="_blank">link text</a>

Absolute URLs vs. Relative URLs

위의 두 예제는 절대 URL(전체 웹 주소)를 사용하고 있다.

로컬 링크(동일한 웹 사이트 내의 페이지에 대한 링크)는 상대 URL("https://www"부분 제외)로 지정된다.

Absolute URLs

<p><a href="https://www.w3.org/">W3C</a></p>
<p><a href="https://www.google.com/">Google</a></p>

Relative URLs

<p><a href="html_images.asp">HTML Images</a></p>
<p><a href="/css/default.asp">CSS Tutorial</a></p>

HTML Links - Use an Image as a Link

이미지를 이용해 링크를 사용하려면 <a>태그 안에 <img>태그를 사용하면 된다.

예시 -

<a href="default.asp">
<img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">
</a>

Link to an Email Address

href 속성 안에 mailto를 사용하면 email 프로그램을 실행 시킬 수 있다.

<a href="mailto:someone@example.com">Send email</a>
