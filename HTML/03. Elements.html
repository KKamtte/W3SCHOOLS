HTML Elements

HTML elements는 시작태그, 요소, 닫는태그로 정의되어있다.

<tagname>Content goes here...</tagname>

HTML의 모든 요소는 시작태그와 닫는태그가 있다:

<h1>My First Heading</h1>

<p>My first paragraph.</p>

Start tag Element content End tag
<h1> My First Heading </h1>
<p> My first paragraph. </p>
<br> none none

Nested HTML Elements

HTML의 요소들은 중첩될 수 있다. 모든 HTML의 문서는 HTML요소들의 중첩으로 구성되어있다.

Never Skip End Tag

몇몇 HTML elements는 end tag를 쓰지않더라도 바로 나타난다.( ex)<p>)

Empty HTML Elements

HTML elements 중 content가 없는 것은 empty elements라고 한다. <br>태그의 경우 줄 바꿈으로 정의되어있고, 닫는 태그가 없는 빈 요소이다.

This is a
paragraph with a line break.

HTML is Not Case Sensitive

HTML 태그는 대소문자를 구분하지 않는다. <P>와 <p>는 동일한 의미를 갖는다.
HTML표준 문서의 경우 대소문자 구분을 요구하지않지만 W3C는 HTML에서 소문자를 권장하고 XHTML과 같은 더 엄격한 문서 유형에는 소문자를 요구한다.
