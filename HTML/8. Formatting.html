HTML Text Formatting

HTML에서 특별한 의미를 표현하기 위한 몇 가지 요소가 있다.

This text is bold

This text is italic

This is subscript and superscript

HTML Formatting Elements

  • <b> - Bold text

  • <strong> - Important text

  • <i> - Italic text

  • <em> - Emphasized text

  • <mark> - Marked text

  • <small> - Smaller text

  • <del> - Deleted text

  • <ins> - Inserted text

  • <sub> - Subscript text

  • <sup> - Superscript text

HTML <b> and <strong> Elements

This text is bold
This text is important!

텍스트에 대해 실제로도 강조하고자 하는 경우 strong태그를, 단순히 굵게만 보이도록 하려면 b태그를 사용하면 된다.

브라우저 또는 운영체제에서의 스크린 리더(Screen Reader)를 사용하는 경우 음성 합성(Speech Synthesizer)도구가 페이지를 해석하고 읽어낼 때 strong 태그에 대해 거센 억양으로 음을 낼 수 있도록 하여 실제로 말할 때의 강조를 하듯이 한다.

HTML <i> and <em> Elements

This text is italic
This text is emphasized

텍스트에 대해 실제로도 강조하고자 하는 경우 em태그를, 단순히 기울려서 보이도록 하려면 i태그를 사용하면 된다.

HTML <small> Elements

small 요소는 텍스트가 작게 표현된다.

This is some smaller text.

HTML <mark> Elements

mark 요소는 텍스트에 형광색 표시를 한다(default 값 형광색 style 조정 가능)

Do not forget to buy milk today.

Do not forget to buy drinks today.

HTML <del> Elements

del 요소는 문자의 삭제를 보여준다.

My favorite color is blue red.

HTML <ins> Elements

ins 요소는 문자의 추가를 보여준다. 브라우저는 일반적으로 삽입 된 텍스트에 밑줄을 긋는다.

My favorite color is blue red.

HTML <sub> Elements

sub 요소는 아래 첨자 텍스트로 H2O와 같은 화학식에 사용될 수 있다.

This is subscripted text.

HTML <sup> Elements

sup 요소는 위 첨자 텍스트로 WWW[1]와 같은 각주에 사용될 수 있다.

This is superscripted text.
